# Bank Management System

A Java-based Bank Management System developed using Core Java, JDBC, and MySQL.
This project simulates real-world banking operations with secure authentication
and multiple transaction features.

## Features
- Secure Login and Multi-step Signup
- Deposit and Withdrawal
- Fast Cash
- Balance Inquiry
- Mini Statement
- Transaction History
- PIN Change
- MySQL Database Integration using JDBC

## Tech Stack
- Java
- MySQL
- JDBC
- Apache NetBeans

## Objective
To understand object-oriented programming concepts, JDBC connectivity,
and real-world banking workflows using Java.
